자체 카메라 시스템 — 칸 안을 위에서 본다

테슬라 자율주행이 도로를 위에서 본 평면(BEV)으로 인식하듯, 우리는 지하철 칸 내부를 같은 방식으로 본다. 개인 식별 정보 일체 만들지 않음 (사람 = 점, 가방 = 점).

실시간 카메라 분석 화면
YOLO11 인공지능 + 물체 추적 + 평면 보정 + 3D 시각화 — 분당 8.5 프레임 처리

5단계 처리 흐름

STEP 1
📸 카메라 입력
IP 카메라 또는 브라우저 카메라 → 화상 영상 전송
STEP 2
🎯 인공지능 인식 + 추적
YOLO11 (사람·자전거·차량·가방 등 6종) + BoT-SORT 추적 (같은 사람 연속 추적, ID는 일회성)
STEP 3
📐 평면 보정 (호모그래피)
기울어진 카메라 영상 → 위에서 내려다본 평면 좌표로 변환
STEP 4
📊 점 단위 집계 (개인 식별 X)
사람 위치만 점으로 카운트. 얼굴/지문/번호 등 개인 식별 데이터 일체 미생성
STEP 5
📡 분 단위 broadcast
웹소켓으로 시민 웹앱·네이티브앱·운영자 콘솔 동시 송출

🛡️ 프라이버시 우선 설계

개인 ID 일체 미생성. 점만 보낸다. 차내 진입 행정 절차 없이도 운영 가능한 유일한 방식.

6대 사회적 가치 IDEA — 한 시스템으로 동시 보호

분 단위 칸 안 시각화는 수단일 뿐. MetroEyes 의 진짜 목적은 사회적 약자와 안전 사각지대를 시스템적으로 메우는 것.

IDEA-7
👶

임산부 배려석 자동 감지

칸 양 끝 분홍 좌석 영역을 카메라로 식별. 30초+ 점유 + 임산부 동반자 미감지 → 일반인 점유 의심 자동 알림.

임팩트: 회피율/양보율 정량 데이터 = 서울시 최초 가능. 차내 디스플레이 비강제 안내.
IDEA-안전
🚨

응급 골든타임 자동 검출

한 사람이 30초 이상 정지 → 응급 의심. 시스템이 자동으로 가장 가까운 AED(자동심장충격기) 위치 + 거리 운영자에게 알림.

임팩트: 평균 발견 3분 → 30초 이내. 골든타임 4분 안에 대응 가능.
IDEA-8
⚠️

이태원 참사 사전 경고

에스컬레이터·환승 통로 같은 좁은 영역에서 인구 밀도 + 평균 이동 속도 0.3m/s 이하 + 45초 지속 → 병목 자동 검출.

임팩트: 다중 영역 동시 감지로 군중 밀집 사전 경고. 옆 출구 권장 안내.
IDEA-분실
🎒

분실물 자동 추적

가방 클래스 추적 (BoT-SORT). 사람 없이 가방만 12초 이상 무인 상태가 되면 분실물 의심 자동 broadcast.

임팩트: 종착역 분실물 통합 시스템 즉시 연계. 환승 통로 우산도 추적.
IDEA-무임
💰

무임승차 감지

승강장 카메라로 승객 카운트 vs 게이트 태그 카운트 비교. 차이 발생 시 무임승차 의심 자동 알림.

임팩트: 운영사 매출 손실 회복. 게이트 태그 미스캔 자동 통계.
IDEA-9
🔔

5중 도착 알림 (노이즈 캔슬링)

차내 방송 의존성 제거. ① 시각 banner ② 진동(햅틱) ③ 비프음 ④ 4언어 음성 ⑤ 시스템 알림 동시 발사.

대상: 청각장애인 42만 + 노이즈 캔슬링 1,200만 = 1,242만 잠재 사용자. 장애인 차별 금지법 + 교통 약자법 직결.
