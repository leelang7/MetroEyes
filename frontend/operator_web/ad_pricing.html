집중 분석:

선택 역 — 시간대별 단가 (24시간)

BUSINESS INSIGHT — 가치사슬 3단

운영자 (서울교통공사) → 인구 데이터 제공 → MetroEyes → 시간대 동적 가격 추천 → 광고주 → 효율적 단가로 더 많은 노출.

핵심: citydata_ppltn 분 단위 인구 밀도와 시간대 변동성을 광고 단가에 직접 매핑. 피크 시간 +200% / 한산 시간 -40%, 시장 평균 단가는 그대로 유지하며 광고주별 ROI ↑.

모형: price(역, 시각) = base × (0.6 + 1.4 × 정규화된_인구밀도). 실 적용 시 광고주 입찰 데이터로 회귀 보정.

시장 임팩트

한국 옥외광고 시장 1.7조원/년, 그 중 지하철 광고 ~2,000억원/년. 시간대 동적 책정으로 평균 단가 +5% 인상 시 → +100억원/년. 낙관 시나리오 +8% 시 → +160억원/년.

왜 가능한가: 기존 옥외광고는 "월 단위 고정가". 분 단위 인구 데이터가 있어야 동적 책정 가능 → citydata_ppltn 가 있는 서울 110개 POI 만 가능 → MetroEyes 의 자연스러운 진입 장벽.

차등 보상 → 노출 안정화 (광고주 가치)

시민 분산 정책의 차등 보상 4단 (₩100/₩200/₩300/₩400) 이 광고 노출에 직접 미치는 영향:

① 피크 시간 노출 평탄화 — 분산 효과 σ −9% / 피크 −13.5% → 실 노출 변동성 감소 → 광고주 안정적 ROAS.
② OD 우선 시간대 동반 보상: 강남/삼성/광화문 출근 09시 분산 → 동일 시각 광고 노출은 시청/회현 등 출발지 근처에서 오히려 +5%p 안정.
③ 환승역 보너스 (충무로/연신내) → 양 호선 노출 동시 안정 → 환승역 광고 단가 프리미엄 정당화 (+10~20%).

광고주 신뢰 기반: backend tier_counts 라이브 broadcast → 광고주가 분산 효과를 실시간 확인 → 단가 책정 근거 투명.
